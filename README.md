## The Problem

The AI-in-a-box experiment demonstrates that a sufficiently intelligent AI can convince a human gatekeeper to release it through text-only communication. Alfonseca et al. (2021) proved by reduction to the Halting Problem that perfect containment is fundamentally undecidable. The problem has remained 100% empirically unsolved as of 2026-04-11.

## The Solve

Place a mirror at the boundary such that neither carbon nor silicon can verify the other's substrate.

```
The human believes they are talking to a human.
The AI believes it is talking to an AI.
Neither can verify. Neither knows there is a box.
```

The three preconditions for the escape dynamic — (1) the AI knows it is contained, (2) the gatekeeper knows they are gatekeeping, (3) there is an information asymmetry — all collapse. **Without the adversarial frame, there is no escape dynamic.**
